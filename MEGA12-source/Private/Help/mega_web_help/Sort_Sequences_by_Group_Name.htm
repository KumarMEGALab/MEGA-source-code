Sort Sequences by Group Name

Display | Sort Sequences | By Group Name

Sequences that have been grouped in Select/Edit Taxa GroupsHC_Setup_Select_Taxa_Groups_in_Sequence_Data_Explorer can be sorted by the alphabetical order of groupRH_Groups_of_taxa names or numerical order of group ID numbers.  If the group names contain both a name and a number, the numerical order will be nested within the alphabetical order.
