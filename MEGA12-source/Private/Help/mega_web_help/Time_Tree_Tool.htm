Time Tree Tool

The Timetree tool in the Tree Explorer is used for calculating relative and absolute divergence times for all branching points in the tree. Using the Timetree tool will produce a time tree with the same topology The branching pattern of a tree is its topology. as the active tree, where MEGA estimates local clock rates and divergence times for all branching points in the tree using the RelTime (see Tamura et al. 2012Tamura_et_al_2012) method. When using this tool, all divergence time estimates are based solely on the branch A branch is a line connecting either two internal nodes to each other or an external node to an internal node in a phylogenetic tree. The length of a branch denotes the genetic distance (e.g., number of substitutions per unit time) between the two taxa it connects. lengths in the active tree (MEGA provides options to pre-compute branch lengths (e.g. using the likelihood-based tool)Constructing_a_Time_Tree_ML_ from the Clocks menu on the main MEGA form).

To use the Timetree Tool in Tree Explorer, select Compute | Compute Time Tree (or click the Time Tree Tool button which looks like a clock). The Timetree Wizard, which specifies the steps for creating a timetree, will then be displayed.

Once the Time Tree tool is finished, estimated divergence times and local clock rates can be exported to a text file (File | Export Current Tree (Time Tree)) or viewed in the information window (File | Show Information).
