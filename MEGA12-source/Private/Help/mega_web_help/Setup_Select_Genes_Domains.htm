The Setup/Select Genes & DomainsHC_Genes_Domains_Dialog dialog box allows you to view, specify, and edit genes and domains A domain is a continuous block of sites in a sequence alignment. A domain can be free-standing or assigned to genes and protein-coding (e.g., exons) or non-coding (e.g., introns). Domains can be defined in the input data, and can be defined and edited in the Setup Genes Domains dialog. and to label sites.
