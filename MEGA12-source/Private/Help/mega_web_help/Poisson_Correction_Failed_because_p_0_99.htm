Poisson Correction Failed because p > 0.99

For an amino acid estimation of distances, the proportion of amino acids that differ between two sequences has exceeded 99% and the Poisson correction distance formula cannot be applied. If you wish to know which pair(s) of sequences has this problem, use the Distances|PairwiseHC_Pairwise_Distances option.  All such pairs will be shown in the Distance Matrix Dialog with a red n/c (not computable).
