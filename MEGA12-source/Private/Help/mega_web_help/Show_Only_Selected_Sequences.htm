Show Only Selected Sequences

Display | Show only Selected Sequences

The check boxes in the left column of the display grid can be used to select or deselect sequences for analysis.  Subsequent use of the “Show Only Selected Sequences” option in the Display menuHC_Display_Menu_in_Sequence_Data_Explorer of Sequence Data ExplorerHC_Sequence_Data_Explorer hides all the deselected sequences and displays only the selected ones.
