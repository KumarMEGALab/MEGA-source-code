Setup/Select Genes & Domains (Sequence Data Explorer)

Data | Setup/Select Genes & Domains A domain is a continuous block of sites in a sequence alignment. A domain can be free-standing or assigned to genes and protein-coding (e.g., exons) or non-coding (e.g., introns). Domains can be defined in the input data, and can be defined and edited in the Setup Genes Domains dialog.

Setup/Select Genes & DomainsHC_Setup_Genes_Domains, can be invoked from within the Data menuHC_Data_Menu_in_sequence_Data_Explorer in Sequence Data Explorer,HC_Sequence_Data_Explorer and is also available in the main interface directly in the Data Menu.
