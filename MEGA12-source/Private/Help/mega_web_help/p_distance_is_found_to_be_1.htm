The Kimura (1980) distance correction is used in a number of operations, including calculating nucleotide distances and synonymousRH_Synonymous_change and nonsynonymousRH_Nonsynonymous_change substitution distances.  These formulas cannot be applied if the argument in the logarithm approaches zero or becomes negative.  If you see this error message, then this has happened for one or more pairs in your data.  If you wish to know which pair(s), use the Distances|PairwiseHC_Pairwise_Distances  option.  All such pairs will be shown in the Distance Matrix Dialog with a red n/c.
