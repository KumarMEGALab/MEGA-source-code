Convert to MEGA Format

File | Convert File Format to MEGA

This item allows you to choose the file and/or the format that you would like to use to convert a given sequence data file into a MEGA format. It converts the data file and displays the converted data in the editor.

Files written in a number of popular data formats can be converted into MEGA format. MEGA supports conversion of CLUSTAL, NEXUS (PAUP, MacClade), PHYLIP, GCG, FASTA, PIR, NBRF, MSF, IG, and XML formats. Details about how MEGA reads and converts these file formats are given in the section Importing Data from Other Formats.
