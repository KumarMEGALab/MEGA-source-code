Newick Format

NEWICK is a simple format used to write out trees in a text file.  While this is a hard-to-read format for humans, it is very useful for exchanging trees between different types of software. An example of the contents of a NEWICK format tree file is given below (note that semi-colon is needed to end the tree).  Further information on this format can be found at Joe Felsensteinâ€™s website!Inet("http://evolution.genetics.washington.edu/phylip/felsenstein.html").

((raccoon, bear),((sea_lion,seal),((monkey,cat), weasel)),dog);
