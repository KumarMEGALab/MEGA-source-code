No. of differences (Nucleotide)

 

This distance is the number of sites at which the two compared sequences differ.  If you are using the pairwise deletion optionRH_Pairwise_deletion_option for handling gaps and missing data, it is important to realize that this count does not normalize the number of differences based on the number of valid sites compared, if the sequences contain alignment gaps Phylogenetic analysis on two or more DNA or amino acid sequences requires that the sequences be aligned so that the substitutions can be accurately enumerated. During alignment, gaps must be introduced in sequences that have undergone deletions or insertions. These gaps are known as alignment gaps or indels..  Therefore, we recommend that if you use this distance you use the complete-deletion Option.

 

For this distance, MEGA provides facilities for computing the following quantities:

d: Transitions + Transversions: Number of different nucleotide sites.

s: Transitions only: Number of nucleotide sites with transitional differences.

v: Transversions only: Number of nucleotide sites with transversional differences.

R = s/v: Transition A transition occurs when a purine is substituted by a purine, or a pyrimidine by a pyrimidine./transversions ratio.

L: No of valid common sites: Number of compared sites.

Formulas for computing these quantities and their variances are as follows.

Var(d)  = nd(L - nd)/L

Var(s) = s(L - s)/L

Var(v) = v(L - v)/L
