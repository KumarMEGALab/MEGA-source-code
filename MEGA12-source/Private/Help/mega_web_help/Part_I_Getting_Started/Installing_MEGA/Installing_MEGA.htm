Installing MEGA

Windows Operating Systems

The preferred way to install MEGA is to download the installer directly from the website (www.megasoftware.net). Then run the installation program which will install MEGA in the location (directory) you specify.

If you are experiencing problems during installation, we recommend you disable the UAC (User Account Control). This can be done through Control Panel -> User Accounts -> Change Security Settings -> Deselect the checkbox (Use user account control (UAC)) and click ok. You may also need to temporarily disable your anti-virus software if it interferes with the installation.

We recommend that you install MEGA in one of the ways described above. Please do not simply copy MEGA-related files from one computer to another, as MEGA may not work properly if installed in this manner.

If you are installing MEGA on a set of lab/classroom computers, you may use whatever method most commonly used for your type of deployment, but we may not provide support.

***Note for administrators*** If you need to perform a mass installation of MEGA, the installer can be run from a batch script or command line using the /SILENT /VERYSILENT flags. Then MEGA will be installed with default options. For example:

 MEGAX.X.X_setup.exe /SILENT /VERYSILENT

 

Mac OS X Systems

Download the MEGA .pkg file from the website (http://www.megasoftware.net). Double-click the .pkg file and follow the installer wizard instructions.

 

Linux Systems (Debian)

Download the MEGA .deb package from the website (http://www.megasoftware.net). Then install MEGA using the dpkg command:

sudo dpkg -i path/to/the/deb/package

 

Linux Systems (Redhat-like)

Download the MEGA .rpm package from the website (http://www.megasoftware.net). Then install MEGA using the rpm command:

sudo rpm -ivh --replace-files path/to/the/rpm/package
