Equal Input Model (Gamma)

In real data, amino acid frequencies usually vary among the different kinds of amino acids and substitution rates are not uniform among sites. In this case, the correction based on the equal input model gives a better estimate of the number of amino acidsubstitutions than the Poisson correction distance.  The rate variation among sites is modeled using the Gamma distribution; for computing this distance you will need to provide a gamma parameterRH_Gamma_parameter (a).
