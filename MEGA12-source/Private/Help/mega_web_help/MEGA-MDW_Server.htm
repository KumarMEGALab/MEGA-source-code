MEGA-MDW Server

All EvoD, PolyPhen-2, and SIFT predictions are pre-computed and stored on the MEGA-MDW web server. For all variants of interest, predictions of functional impact and related data are retrieved from the MEGA-MDW web server and displayed by the MEGA-MD rich graphical user interface (GUI).

The MEGA-MDW server can also be accessed directly through its web interface, although it does not provide the same rich functionality that is found in the MEGA-MD desktop client application. However, for large numbers (e.g. > 10,000) of nsSNVs, the MEGA-MDW Server web interface may be more suitable than the MEGA-MD client application (depending on your internet connection speed) as the retrieval of data for many nsSNVs may take some time. The MEGA-MDW server can be accessed from any web browser at www.mypeg.info/evod .
