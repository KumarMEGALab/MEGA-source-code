Labeled Sites

Sites in a sequence alignment can be categorized and labeledRH_Site_Label with user-defined symbols. Each category is represented by a letter or a number. Each site can be assigned to only one category, although any combination of categories can be selected for analysis.

Labeled sites work independently of and in addition to genesRH_Gene and domains A domain is a continuous block of sites in a sequence alignment. A domain can be free-standing or assigned to genes and protein-coding (e.g., exons) or non-coding (e.g., introns). Domains can be defined in the input data, and can be defined and edited in the Setup Genes Domains dialog.RH_Domains, thus allowing complex subsets of sites to be defined easily.
