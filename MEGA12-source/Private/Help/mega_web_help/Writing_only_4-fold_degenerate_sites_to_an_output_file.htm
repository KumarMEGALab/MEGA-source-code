Writing only 4-fold degenerate sites to an output file

All sequence data subset facilities are accessible through the Export DataHC_Export_Data_in_Sequence_Data_Explorer command in the Sequence Data Explorer.  To write 4-fold degenerateRH_Degeneracy sites to a file, highlight the 4-fold degenerate sites on the screen and then select Export DataHC_Export_Data_in_Sequence_Data_Explorer.  In that command, choose to write only the highlighted sites.  For example, if you select to write only the third codon A codon is triplet of nucleotides that codes for a specific amino acid.RH_Codon positions, all 4-fold degenerate sites found in the third codon A codon is triplet of nucleotides that codes for a specific amino acid. positions will be written to the file.
