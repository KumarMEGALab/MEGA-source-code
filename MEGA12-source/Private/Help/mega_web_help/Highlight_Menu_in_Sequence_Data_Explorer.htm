Highlight Menu (in Sequence Data Explorer)

This menu can be used to highlight certain types of sites.  The options are constant sitesRH_Constant_site, variable sitesRH_Variable_site, parsimony-informative sitesRH_Parsimony_informative_site, singleton sitesRH_Singleton_Sites, 0-foldRH_Degeneracy, 2-foldRH_Degeneracy and 4-fold degenerate sitesRH_Degeneracy.
