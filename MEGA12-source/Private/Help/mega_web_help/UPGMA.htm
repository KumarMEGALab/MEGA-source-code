UPGMA

This method assumes that the rate of nucleotide or amino acid substitution is the same for all evolutionary lineages. An interesting aspect of this method is that it produces a tree that mimics a species tree, with the branch A branch is a line connecting either two internal nodes to each other or an external node to an internal node in a phylogenetic tree.
