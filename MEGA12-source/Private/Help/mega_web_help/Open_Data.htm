Open Data

Data | Activate Data for Analysis | Get Data from MEGA text File

Choose this command to load a data file for analysis. A dialog box will appear to allow you to give the data file name. MEGA will first read the data file to check if it contains the Format commandRH_Format_command (see MEGA formatRH_MEGA_Format), which specifies certain attributes of the input data (e.g., type of data).  If MEGA does not find sufficient information in the format command, it will request the necessary information through an Input Data FormatHC_Input_Data_Format dialog. 

If you attempt to open a dataset from a file and MEGA detects inconsistencies or errors in the format, it will open the file in the text editorHC_Text_Editor, allowing you to make changes in the text file so that it conforms to the MEGA format.

Once a data file is opened successfully, the Sequence Data ExplorerHC_Data_Explorer>Main icon will be displayed in the main windowHC_Main_MEGA_Window>Second.
