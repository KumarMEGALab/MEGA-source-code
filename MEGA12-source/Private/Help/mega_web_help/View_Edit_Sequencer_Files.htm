Align | Edit/View Sequencer Files (Trace)…

Use this option to view/edit the sequence data in ABIRH_ABI (*.abi and .ab1) and Staden The Staden file format is used to store data from DNA sequencing instruments. Each file contains the data for a single reading and includes the called sequence as well as additional data obtained from the reading. This file format was first described in Dear, S and Staden, R. “A Standard file format for data from DNA sequencing instruments”, DNA Sequence 3, 107-110, (1992) MEGA is able to display the contents of a Staden-formatted trace file using MEGA’s Trace File Editor, which is part of the Alignment Explorer.RH_Staden (.scf) files.  The Alignment ExplorerHC_Alignment_Builder provides this option directly.
